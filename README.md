# BeatMe

> **One question. Once a day.**
>
> *"Did I beat myself today?"*

BeatMe is a local-first desktop reflection application built to encourage daily consistency rather than productivity obsession.

Instead of tracking dozens of habits, tasks, or metrics, BeatMe asks a single question at the end of every day and visualizes whether you stayed true to your own standards.

---

## ✨ Features

- 🌙 Daily reflection journal
- 🔥 GitHub-style consistency heatmap
- 📊 Streaks and statistics
- 💾 Local SQLite database
- 🖥️ Native desktop application (Tauri)
- 🚀 Automatic startup with the operating system
- 📴 Completely offline
- 🔒 No accounts. No cloud. No tracking.

---

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** Rust (Tauri v2)
- **Database:** SQLite
- **State Management:** Zustand

---

## Philosophy

BeatMe isn't designed to maximize productivity.

It's designed to maximize honesty.

Every night you answer one simple question:

> **Did I beat myself today?**

No social features.
No leaderboards.
No dopamine mechanics.

Just one honest answer.

---

## Getting Started

```bash
npm install
npm run tauri dev
```

Build production:

```bash
npm run tauri build
```
